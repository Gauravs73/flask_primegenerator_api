
This API is build using Flask framework of python 


To use this API data input must be given as raw JSON data format

You can use Postman app for verification and working of this API
by providing JSON data input to the API

steps:

install Flask on your system and python ide like pycharm
import this file as a project and run main.py file in local server
while opening in browser add "/primegen" route to the http address of local server
"/primegen" is route for function to normal method of prime number generation

similarly to access  prime sieve method of prime number generation add "/sieve" at
the end of http address of local server 

copy this local http address and post it over postman with providing JSON format input
as the two numbers
